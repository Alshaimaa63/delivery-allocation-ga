# Simple Delivery Allocation 🚚⚖️  

  A hybrid AI system that combines **Breadth-First Search (BFS)** for shortest path calculation and **Genetic Algorithms (GA)** for delivery load balancing between two vehicles.


## Features

- BFS: Finds the shortest path distance from the depot to delivery points while avoiding obstacles.
- Genetic Algorithm: Optimizes package allocation using:
  - Selection
  - Crossover
  - Bit Flip Mutation
- Load Balancing: Minimizes the distance difference between Vehicle 1 and Vehicle 2.
- Visualization: Displays depot, obstacles, and delivery assignments using Matplotlib.

## Results

- Balanced delivery distribution between both vehicles.
- Grid-based visualization with obstacles and assigned points.
- **Language:** Python (Matplotlib).
